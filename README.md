## Description

Utility for self-documented Makefile
It shows rules in Makefiles with documents.

## Installation

    % go get github.com/Songmu/make2help/cmd/make2help

## Synopsis

    % make2help [Makefiles]

## Options

```
-all                display all rules in the Makefiles
```
